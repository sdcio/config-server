## enable profiling

Enabled pprof by default

```
kubectl port-forward pod/<pod> -n network-system 8081:8081

curl -s "http://127.0.0.1:8081/debug/pprof/profile" > ./cpu-profile.out

go tool pprof -http=:8080 ./cpu-profile.out
```
